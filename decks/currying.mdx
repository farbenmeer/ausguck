lodash.curry

```js
import {curry} from 'lodash.curry';

const abc = (a, b, c) => {
    return [a, b, c];
};

const curried = curry(abc);

curried(1)(2)(3);
// => [1, 2, 3]
 
curried(1, 2)(3);
// => [1, 2, 3]
 
curried(1, 2, 3);
// => [1, 2, 3]
```

---

If you apply the same arguments to a function more than once you should think if currying would be a good choice for you.

How does currying looks like in other languages?

---

Go:

```go
func add(x int) func(int) int {
    return func(y int) int {
        return x + y
    }
}

func main() {
    fmt.Println(add(1)(2)) // 3
    add1 := add(1)
    fmt.Println(add1(2)) // 3
    fmt.Println(add1(3)) // 4
}
```

---

Haskell:

```hs
add x y  = x + y 

add1 = add 1

main = do
    print $ add 1 2 -- 3
    print $ add1 2 -- 3
    print $ add1 3 -- 4
```
